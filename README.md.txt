# Influencer Fraud Detection

## Project Overview

Influencer marketing has become a key strategy for businesses, but fake engagement and fraudulent influencer accounts can lead to poor marketing decisions and financial losses. This project applies machine learning-based anomaly detection techniques to identify potentially fraudulent influencers by analyzing engagement metrics and profile statistics.

The solution generates fraud scores that help businesses identify suspicious influencer accounts and support more informed influencer selection.

---

## Business Problem

Brands often collaborate with influencers based on follower count and engagement metrics. However, fake followers, artificial likes, and manipulated engagement can make these metrics misleading.

This project aims to detect abnormal engagement patterns and identify potentially fraudulent influencer accounts using unsupervised machine learning techniques.

---

## Objectives

- Detect suspicious influencer profiles using anomaly detection.
- Analyze engagement-related metrics to identify unusual behavior.
- Perform data preprocessing and feature engineering.
- Generate fraud scores for influencer accounts.
- Support data-driven decision-making for influencer marketing campaigns.

## Machine Learning Approach

This project uses **Isolation Forest**, an unsupervised anomaly detection algorithm designed to identify unusual data patterns without requiring labeled training data.

The workflow includes feature scaling using **StandardScaler**, followed by anomaly detection to classify influencer profiles based on engagement behavior. Suspicious profiles are assigned fraud indicators to support further analysis.
